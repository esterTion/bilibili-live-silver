# bilibili-live-silver
自动获取B站直播瓜子

使用
------

    Win需要cygwin环境
    
    ./live-silver <access-key>
    
    使用需要个人access-key，可以于 https://www.biliplus.com/api/login.php 登录后从cookie中提取
